# Text-Enhanced Portfolio RL

Scaffold for the project structure described in `axiom_tilt.md`.

## Overview
This repository is organized around a staged pipeline:
1. ingest and align market and text data,
2. build FinBERT-based text representations,
3. engineer quantitative features,
4. train stock scoring models,
5. learn weekly policy actions,
6. optimize implementable portfolios,
7. backtest and report results.

## Quick Start
Create the environment, fill in the configs under `configs/`, and implement the modules in `src/` stage by stage.
